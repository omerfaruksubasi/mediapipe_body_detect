## Kurulum

1. **Depoyu klonlayın:**

    ```bash
    git clone https://github.com/kullanici-adiniz/proje-adi.git
    ```

2. **Gerekli kütüphaneleri yükleyin:**

    ```bash
    pip install -r requirements.txt
    ```

3. **Mediapipe'yi yükleyin:**

    ```bash
    pip install mediapipe
    ```
